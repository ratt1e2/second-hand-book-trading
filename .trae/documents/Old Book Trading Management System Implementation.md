# 旧书交易管理系统实现计划

* 配置 `pom.xml` 依赖（严格遵循提供的配置）

* 搭建项目基础结构（controller、service、mapper、entity 等）

### 管理端前端 (Vue 3)

* 使用 Vite 创建 Vue 3 项目

* 配置 `package.json` 依赖（严格遵循提供的配置）

* 搭建项目基础结构（components、views、router 等）

### 移动端 (微信小程序)

* 创建微信小程序项目结构

* 配置必要的页面和组件

## 2. 数据库设计与实现

### 核心数据表设计

* **用户表** (`user`)：用户信息、微信OpenID、手机号等

* **书籍表** (`book`)：书籍信息、ISBN、价格、新旧程度、状态等

* **分类表** (`category`)：书籍分类信息

* **订单表** (`order`)：订单信息、状态、金额等

* **订单详情表** (`order_item`)：订单商品详情

* **地址表** (`address`)：用户收货地址

* **评论表** (`review`)：书籍评论信息

* **轮播图表** (`carousel`)：首页轮播图

### 生成 DDL 语句

* 为所有表生成包含中文注释的建表语句

* 配置合适的索引和外键关系
